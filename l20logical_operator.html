논리 연산은 비트연산과 비슷하다

  • || Or : 논리의 합
    • true || true : true
    • true || false : true
    • false || true : true
    • false || false : false
    • **논리 합은 한 개라도 true 면 모두 true다.
  • && And : 논리의 곱
    • true && true : true
    • true && false : false
    • false && true : false
    • false && false : false
    • **논리 곱은 한 개라도 false 면 모두 false다.

논리연산과 멈추는 지점

js 엔진은 논리 연산에 참 또는 거짓이 나올거라는 확신이 있다면 다음을 하지 않는다.

논리 연산의 우선 순위

곱하기>더하기 a || b && c || b && d ==> a || (b && c) || (b && d)

true || true && false || true && false : true

a && b || c && b || d = > (a && b) || (c && b) || d

true && false || true && true || false : true
